Proyecto EL3313 - Pong FPGA Maestro

Estado funcional:
- VGA estable con doble buffer.
- Pong local funcionando.
- Multijugador SPI funcionando.
- SW15 selecciona modo local/multijugador.
- En modo multijugador, P2 se controla desde la FPGA esclava.
- DDR2 probada y usada desde firmware.
- Imagen estable sin parpadeo fuerte.

Contenido:
- HDL: src/rtl/
- IP personalizado de video: ip/video_vram_axi_core/
- Firmware: firmware/
- Scripts TCL: scripts/tcl/
- Scripts de debug/programación: scripts/tcl/debug/
- Bitstream y ELF precompilados: artifacts/

Para programar bitstream:
vivado -mode batch -source scripts/tcl/debug/program_master_3406.tcl

Para cargar firmware:
xsdb scripts/tcl/debug/load_master_elf.tcl

Nota:
Si se copia el proyecto a otra ruta, puede ser necesario ajustar rutas absolutas dentro de los scripts TCL.
